# Sergey Panfilov — Portfolio

Bilingual personal portfolio focused on AI automation, Python backend development and system analysis.

## Local preview

No build step is required. Run any static server from the repository root:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.
