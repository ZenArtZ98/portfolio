# Sergey Panfilov — Portfolio

Bilingual personal portfolio focused on AI automation, Python backend development and system analysis.

## Included

- English/Russian language switcher and dark/light theme
- Interactive developer console
- LifeOS and Ping Hosts engineering case studies
- Verified code excerpts and architecture maps
- Role-specific downloadable CV files
- Open Graph social preview and search metadata

## Local preview

No build step is required. Run any static server from the repository root:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

The repository is designed for GitHub Pages at `https://zenartz98.github.io/portfolio/`.

Publish from the `main` branch and repository root in **Settings → Pages**.
